# NOIRÉ — Minimalist Luxury E-Commerce
A modern luxury fashion e-commerce website built with Next.js and Sanity CMS.
## 🔗 Live Demo
[View Live Site](https://noire-e-commerce.vercel.app/)
## ✨ Features
- Minimalist luxury design
- Product listing with categories (Women, Men)
- Shopping cart functionality
- Sanity CMS for content management
- Fully responsive design
- TypeScript for type safety
## 🛠️ Tech Stack
- **Framework:** Next.js 14
- **CMS:** Sanity
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
## 🚀 Getting Started
Clone the repo:
git clone https://github.com/a01289507-commits/Noire-E-commerce.git
Install dependencies:
npm install
Run development server:
npm run dev
Open http://localhost:3000
